# 操作备忘录
作为一个新入门的小白，常常会忘记很多非常简单的操作。
但我相信这些差距，可以通过多次实践来弥补。

## 如何上传代码
```bash
# 1. 将修改后的文件添加到暂存区
git add README.md
# 2. 提交更改并添加描述信息
git commit -m "first commit"
# 3. 将本地代码推送到远程 GitHub 仓库的 main 分支。orgin就是git代码时候的地址。
git push origin main
```
命令动作
git add	工作区 → 暂存区	
git commit	暂存区 → 本地仓库
git push	本地仓库 → 远程仓库
